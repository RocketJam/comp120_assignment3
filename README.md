# To get started:
## The command to clone repo
```
git clone git@github.com:RocketJam/comp120_assignment3.git
```
## The command to set upstream
<sub>git remote add upstream git@github.com:RocketJam/comp120_assignment3.git</sub>

## The command to fetch upstream
<sub>git fetch upstream</sub>

## Commands to add, commit & push changes to Read.me
```
git add .
git commit -m "the change message"
git push
```
